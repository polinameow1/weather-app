# Weather App

Приложение для просмотра погоды с возможностью добавления нескольких виджетов. 
## Основные возможности
- Поиск погоды по названию города
- Поиск по координатам (широта и долгота)
- Отображение дополнительной информации: влажность, давлени и тд
- Удаление виджетов по кнопке

## Техническая часть
JavaScript, OpenWeatherMap API, Fetch API для асинхронных запросов, валидация пользовательского ввода

## Использование
- Введите город в поле ввода и нажмите "Добавить виджет погоды"
- Или введите координаты для точного поиска
- Удаляйте ненужные виджеты нажатием на крестик

Приложение по умолчанию показывает погоду в Екатеринбурге.
